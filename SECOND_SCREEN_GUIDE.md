## üñ•Ô∏è How to Use Second Screen Presentation

### **Method 1: Automatic Detection (Recommended)**
1. **Connect your second screen** via HDMI/USB-C/DisplayPort
2. **Extend your desktop** (don't duplicate)
3. **Open the E-Poster app** in your browser
4. **Click on any abstract** - it will automatically detect the second screen and start presentation mode
5. **The presentation will go fullscreen** on your second screen

### **Method 2: Manual Presentation**
1. **Connect your second screen** and extend desktop
2. **Open the E-Poster app** in your browser
3. **Drag the browser window** to your second screen
4. **Click on any abstract** to view it
5. **Click the "Present" button** - it will go fullscreen on the current screen (second screen)

### **Method 3: Windows Display Settings**
1. **Right-click on desktop** ‚Üí Display Settings
2. **Set your second screen as primary** display
3. **Open the E-Poster app** - it will open on the second screen
4. **Click on any abstract** - presentation will start automatically

## üîß Troubleshooting

### **If Second Screen is Not Detected:**
- Make sure you're using **extended desktop** mode, not duplicate
- Try refreshing the page after connecting the second screen
- Use the manual "Present" button instead

### **If Fullscreen Doesn't Work:**
- Make sure your browser allows fullscreen (Chrome/Edge work best)
- Try pressing **F11** for browser fullscreen
- Use the manual presentation mode

### **If Images Don't Load:**
- Check that all files are in the `public/files` directory
- Make sure file names match exactly (case-sensitive)
- Clear browser cache and refresh

## üéØ Best Practices

5. **Use ESC key** to exit presentation mode
